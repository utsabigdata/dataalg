In order to run main.py a firefox geckodriver is needed in the folder. Find the geckodriver here https://github.com/mozilla/geckodriver/releases

To run the data collection run $python main.py
The only options built out so far is Amazon, Product Search.

1. First option select (A) Amazon
2. Type in product your searching for
3. Second option select  (S) Product Search

A firefox window will pop up, your product will be entered into the search bar, then selenium will flip through all the pages.
